# practice.regular_expression

자바스크립트를 기반으로 작성되었습니다.

## 개요

* 정규 표현식은 기호를 사용하여 구현
* 자주 사용하지 않기 때문에 메모, 예제, 검색 활용
* ECMAScript 스펙에 포함되어 있다.
* C++11에서 JavaScript 정규 표현식 채택
* Javascript는 문자열이 중심이다.
* 검색, 치환, test에 활용
* 6개의 method 사용
  * match(), replace(), search(). ,split()
  * exec(), test()
